# Image Gallery Web Page

This project is a simple and responsive Image Gallery web page created using **HTML and CSS**. The gallery displays multiple images in a clean grid layout, allowing users to view images in an organized and visually appealing way. This project helps in understanding layout design, Flexbox/Grid concepts, and responsive web design using basic frontend technologies.

## Features

* Clean and modern gallery layout
* Responsive design for different screen sizes
* Organized image structure using HTML
* Styled using CSS for better visual appearance
* Beginner-friendly project
* Lightweight and fast loading

## Technologies Used

* HTML5
* CSS3

## Project Purpose

The main purpose of this project is to practice and improve frontend development skills, especially in:

* Structuring web pages using HTML
* Styling using CSS
* Creating responsive layouts
* Organizing images in a gallery format
